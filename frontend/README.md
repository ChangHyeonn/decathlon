# 프로젝트 세팅

1. **사용하는 패키지**

   - Vite
   - Node

2. **사용하는 라이브러리 (자세한 내용은 `package.json` 파일 참고)**
   - axios
   - react
   - react-router-dom
   - recharts
   - styled-components
   - zustand

## 실행 방법

1. 현재 디렉토리에서 CMD 또는 터미널 열기
2. `npm install` 또는 `npm i` 명령어 실행
3. `npm run dev` 명령어로 개발 서버 실행
4. 접속 주소: `http://localhost:5173` 또는 터미널에서 확인
